**Data Independence** in a **Database Management System (DBMS)** means:

> **You can change the structure of the database without affecting the application programs that use the data.**

In simple words:

* Changes in **how data is stored** should **not break** the programs that access the data.
* Changes in **table structure** should **not require rewriting application code**.

This is one of the **most important advantages of DBMS over file-based systems**.
