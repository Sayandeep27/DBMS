## ðŸ“Œ What is a View in DBMS?

A **View** in a Database Management System (DBMS) is a **virtual table** created using a SQL query.

* It **does not store data physically**.
* It shows data that already exists in one or more base tables.
* It behaves **like a table** when you query it.

ðŸ‘‰ Think of a **view as a saved SQL query** with a name.
