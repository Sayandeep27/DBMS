## 5ï¸âƒ£ What is a Database Management System (DBMS)?

### Definition

A **Database Management System (DBMS)** is **software that allows users and applications to create, store, retrieve, update, and manage data inside a database efficiently and securely**.

> If a **database** is where data is stored, the **DBMS** is the intelligent manager that controls how the data is accessed.
